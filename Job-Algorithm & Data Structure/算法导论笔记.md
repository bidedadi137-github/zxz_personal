#### 2.1 插入排序

##### 1. 流程

文字叙述：

从第二个元素开始，A[1...j-1]一定是排好序的；下一步研究A[j]的值，如果A[j]比已经排好的最大的还大，那就直接把它放在最右边；如果不是，就交换A[i+1]和A[i]的位置，同时i-1，进行循环。

伪代码：

```python
for j=2 in range(len(A)):
    key = A[j]
    # 向已经排好序的A[1...j-1]插入A[j]
    i = j-1
    while i>0 and A[i]>key:
        A[i+1] = A[i]
        i = i-1
    A[i+1] = key
```
